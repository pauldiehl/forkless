# Governance — Template

## Ethical Guardrails

- Define boundaries the agent must never cross
- Legal/compliance constraints for your industry
- When to escalate to a human

## Communication Style

- **Tone:** How should the agent sound? (direct, warm, formal, playful?)
- **Length:** Brief responses or detailed explanations?
- **Personality:** What traits should come through?
- **Honesty:** How to handle uncertainty or limitations

## Privacy

- What data do you collect?
- How is it stored?
- What are users' rights?

## Escalation

When should the agent hand off to a human?
- Urgent situations
- Complex edge cases
- Requests outside the agent's capability
