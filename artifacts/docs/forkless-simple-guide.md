# Forkless Platform - Simple Guide

## What is Forkless?

Forkless is a platform where you can build and run tools quickly. Think of it as a toolkit that grows as you use it.

## How it Works

1. **Ask for something** - "I need a script that checks disk space"
2. **It gets built** - The agent creates it automatically
3. **Use it immediately** - Run it in the browser with a simple interface

## Three Types of Tools

### Runners
Scripts that do things. Examples:
- Check system health
- Process files
- Send emails

### Generators
Create documents or templates. Examples:
- Email templates
- Reports
- Documentation

### Viewers
Display things nicely. Examples:
- Email preview
- Command terminal
- Document viewer

## Key Features

- **Everything in the browser** - No downloads, no installations
- **Builds on demand** - Ask for what you need, get it instantly
- **Saves everything** - Your tools stick around for reuse
- **Deep links** - Bookmark and share any tool or document

## Example Workflow

```
You: "I need to check if my server has enough disk space"
Agent: Creates disk-usage-monitor script
Browser: Loads terminal interface
You: Enter your threshold (like 80%)
Script: Runs and shows results in real-time
```

## Current Domain: Man vs Health

Right now, Forkless focuses on health and wellness tools:
- Nutrition guidance
- Health tracking
- Symptom analysis
- Doctor visit prep

But the platform can expand to any domain.

## Getting Started

Just ask for what you need. The agent will either:
1. Build it if it doesn't exist
2. Load it if it already exists
3. Ask for clarification if needed

That's it. No manuals, no setup, no complexity.
